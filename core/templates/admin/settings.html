🌐 什么是远程访问？
安装 Tailscale 后，您可以在家里、出差时通过手机或电脑访问公司的 ERP 系统。
完全免费，无需公网 IP，无需复杂的网络配置。适合需要异地办公的团队。

📋 三步开通远程访问

{# Step 1 #}
1
注册 Tailscale 账号

① 打开 tailscale.com，点击右上角 "Get Started"

② 推荐使用微软账号登录（国内网络最顺畅）

③ 免费版支持最多 100 台设备，完全够用

{# Step 2 #}
2
在服务器电脑上安装客户端

① 打开 tailscale.com/download 下载 Windows 版本

② 运行安装程序，一路 Next 即可

③ 安装完成后，右下角系统托盘会出现 Tailscale 图标

④ 右键点击图标 → Log in... → 用第一步注册的账号登录

⑤ 登录成功后会看到一个 100.x.x.x 格式的地址，这就是远程访问地址

{# Step 3 #}
3
在远程设备上访问

① 在需要远程访问的设备（手机/电脑）上也安装 Tailscale

② 用同一个账号登录

③ 在浏览器中打开：

http://100.x.x.x:8000

（将 100.x.x.x 替换为第二步中看到的地址）

④ 把这个地址告诉需要远程访问的同事即可 ✅

{# ── 方案 B: FRP ── #}
🔗 方案 B：FRP 内网穿透 需技术基础
⭐⭐⭐ 中等 · ¥50-100/月 ▸

适合场景：有公网服务器的技术团队，需要固定地址对外提供服务

原理：通过一台有公网 IP 的云服务器做中转，将内网的 CORE ERP 端口暴露到公网

操作步骤：

  1. 购买一台云服务器（阿里云/腾讯云最低配即可，约 ¥50-100/月）
  2. 在服务器上下载并运行 frps（服务端）
  3. 在 CORE ERP 所在电脑上下载并运行 frpc（客户端）
  4. 配置 frpc.toml（示例如下）
  5. 通过 http://你的服务器IP:8000 访问
# frpc.toml 示例配置
serverAddr = "你的云服务器IP"
serverPort = 7000

[[proxies]]
name = "core-erp"
type = "tcp"
localIP = "127.0.0.1"
localPort = 8000
remotePort = 8000
{# ── 方案 C: 云服务器 ── #}
☁️ 方案 C：云服务器部署 最稳定
⭐⭐⭐⭐ 较难 · ¥100-300/月 ▸

适合场景：需要稳定长期运营，多人远程使用

原理：将 CORE ERP 直接部署在云服务器上，通过公网 IP 或域名访问

操作步骤：

  1. 购买云服务器（推荐 Ubuntu 22.04，2核4G 以上）
  2. 安装 Python 3.12 + 上传项目代码
  3. 使用 Gunicorn + Nginx 部署（生产级方案）
  4. 配置域名（可选）和 HTTPS 证书
  5. 通过域名或 IP 直接访问

💡 此方案需要一定的 Linux 运维经验，建议有技术人员支持时选择。
