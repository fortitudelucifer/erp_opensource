{# ── 快速开始 ── #}
管理员如何创建员工账号？▸

有两种方式为员工创建账号：

方式一：管理员直接创建
登录后点击导航栏「用户管理」→ 点击「新建用户」→ 填写信息并选择角色 → 保存。账号立即可用。

方式二：员工自行注册
将邀请码告诉员工 → 员工打开注册页面 → 输入邀请码和个人信息 → 提交后等待管理员审核。

如何查看和设置邀请码？▸

点击导航栏「⚙️ 设置」→ 基本设置 Tab → 页面底部可以看到当前邀请码。

点击「🔄 重新生成」按钮可以随时更换邀请码，旧邀请码立即失效。

提示：邀请码是6位数字，建议通过内部通讯工具私下告知员工。

员工如何注册和登录？▸

注册步骤：

  1. 打开管理员提供的系统地址（如 http://192.168.1.xxx:8000）
  2. 点击「注册新账号」
  3. 输入管理员提供的6位邀请码
  4. 填写个人信息和密码
  5. 提交后等待管理员审核通过

登录：审核通过后，用用户名和密码登录即可。

局域网内的其他电脑如何访问系统？▸

CORE ERP 启动后，黑色控制台窗口会显示「局域网访问」地址，例如：

http://192.168.1.100:8000

同一局域网（WiFi/有线）下的任何设备，在浏览器输入这个地址即可访问。

如果无法访问，请检查 Windows 防火墙是否放行了 8000 端口（安装包会自动配置）。

{# ── 项目管理 ── #}
如何新建合同/项目？▸

登录后点击导航栏「项目/合同」→ 点击「新建合同」→ 填写项目信息（名称、编号、客户、金额等）→ 保存。

新建的项目默认状态为「进行中」，后续可以在项目详情页修改状态。

如何上传文件附件？▸

进入项目详情页 → 点击「文件管理」Tab → 点击「上传文件」→ 选择文件并填写描述 → 上传。

支持常见文件格式（PDF、Word、Excel、图片等），单个文件大小限制为 50MB。

如何给项目分配任务？▸

进入项目详情页 → 点击「任务分配」Tab → 选择负责部门和人员 → 设置截止日期 → 保存。

被分配的人员登录后可以在首页看到自己的待办任务。

如何查看项目进度？▸

首页仪表盘显示核心指标（合同总数、进行中、预警项目等）。

点击「项目/合同」可以看到所有项目列表，支持按状态、日期筛选。

{# ── 用户权限 ── #}
系统有哪些用户角色？▸
admin 系统管理员，拥有所有权限，可管理用户、设置、查看日志
boss 老板/总经理，可查看所有数据和报表
manager 部门经理，管理本部门项目和人员
employee 普通员工，查看和处理分配给自己的任务
如何修改员工角色？▸

仅管理员可操作：「用户管理」→ 找到对应用户 → 点击「编辑」→ 修改角色 → 保存。

角色变更立即生效，用户下次刷新页面时会看到新的权限。

忘记密码怎么办？▸

目前系统不支持自助找回密码。请联系管理员：

管理员操作：「用户管理」→ 找到对应用户 → 点击「编辑」→ 设置新密码 → 保存。

然后将新密码告知该员工即可。

{# ── 系统维护 ── #}
如何备份数据？▸

CORE ERP 使用 SQLite 数据库，所有数据存储在一个文件中：

C:\CoreERP\data\erp.db

备份方法：定期复制 data 文件夹到U盘或网盘即可。

建议：每周至少备份一次，重要操作前也建议备份。

服务无法启动怎么办？▸

排查步骤：

  1. 检查端口占用：确认 8000 端口没有被其他程序占用
  2. 查看服务日志：C:\CoreERP\data\service.log
  3. 尝试手动启动：双击 C:\CoreERP\start.bat 查看错误信息
  4. 重启服务：在设置 → 系统信息 Tab 点击「重启服务」
  5. 如以上都无效，重启电脑后再试
如何更新到新版本？▸

更新步骤：

  1. 备份 data 文件夹（重要！）
  2. 下载新版安装包
  3. 直接运行新版安装包，选择相同安装目录覆盖安装
  4. 安装程序会自动停止旧服务、安装新版、启动新服务
  5. 您的数据会自动保留
如何实现互联网远程访问？▸

请前往「⚙️ 设置」→「远程访问」Tab，我们提供了三种方案的详细教程：

  • Tailscale（推荐）— 最简单，适合小团队
  • FRP 内网穿透 — 需要公网服务器
  • 云服务器部署 — 最稳定，适合长期运营
